INTRODUCTION
------------

This module allows admin to configure guest user registration between
specified time.Is helpful when admin don't want to allow guest user
registration. for more info visit
https://www.drupal.org/project/time_based_registration

REQUIREMENTS
------------

 This module have dependency on date module.

INSTALLATION
------------

* Install as you would normally install a contributed Drupal module. See:
  https://www.drupal.org/documentation/install/modules-themes/modules-7
  for further information.

CONFIGURATION
-------------

* Copy the "Time Based Registration" directory under "modules/" or
  "sites/all/modules";
* Go to "admin/modules" and enable "Time Based Registration" module;
* Go to "admin/config/people/time-based-registration" and configure the
  settings.
